# Compteur de mots Java

## Description

Ce programme Java lit un fichier texte et compte le nombre total de mots.

## Fonctionnement

Le programme :
- ouvre un fichier texte,
- lit chaque ligne,
- sépare les mots avec les espaces,
- compte les mots,
- affiche le total final.

## Technologies utilisées

- Java
- Scanner
- File
